\# 📧 AI Email Generator



\## Description

FastAPI project that generates emails using Groq AI.



\## Features

\- Formal / Friendly / Cold email generation

\- Simple UI using HTML + JS

\- FastAPI backend



\## Run Project

pip install fastapi uvicorn groq python-dotenv



uvicorn main:app --reload



Open:

http://127.0.0.1:8000/ui
